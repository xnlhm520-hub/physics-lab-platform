# 西宁市第二中学物理实验智慧学习平台

## 项目简介

西宁市第二中学2025学业水平合格性考试物理实验操作测试智慧学习平台，是一个基于AI语义匹配的物理实验学习系统。平台提供四个高中物理实验的完整视频教学和智能问答功能，帮助学生更好地掌握实验操作技能。

## 项目结构

```
Acelearn/
├── index.html                    # 主页入口
├── ai-question01-answer.html    # 实验一：探究弹簧弹力与形变量的关系
├── ai-question02-answer.html    # 实验二：验证机械能守恒定律
├── ai-question03-answer.html    # 实验三：练习使用多用电表
├── ai-question04-answer.html    # 实验四：电池电动势和内阻的测量
└── docs/                        # 文档目录
```

## 功能特点

### 🎯 核心功能

1. **实验视频学习**
   - 每个实验提供完整的B站视频链接
   - 支持从指定时间点播放视频片段

2. **AI智能问答**
   - 语义理解匹配：智能分析用户问题
   - 精准定位：快速匹配相关视频片段
   - 关键词匹配：支持多种表达方式的查询
   - 视频片段播放：直接跳转到相关时间点

3. **响应式设计**
   - 支持PC、平板、手机等多种设备
   - 自适应布局，提供良好的用户体验

## 实验内容

### 实验一：探究弹簧弹力与形变量的关系
- **学习目标**：掌握胡克定律的应用，学习数据处理和图像分析方法
- **视频链接**：B站 BV1yVyeBmEES
- **AI问答页面**：`ai-question01-answer.html`

### 实验二：验证机械能守恒定律
- **学习目标**：通过自由落体运动验证机械能守恒定律，掌握实验操作和数据处理方法
- **视频链接**：B站 BV1E61NB8Eyy
- **AI问答页面**：`ai-question02-answer.html`

### 实验三：练习使用多用电表
- **学习目标**：学习多用电表的使用方法，掌握测量电阻、电压和电流的技能
- **视频链接**：B站 BV1fdmqBxEk3
- **AI问答页面**：`ai-question03-answer.html`
- **特点**：包含26个详细视频片段，覆盖多用电表使用的各个方面

### 实验四：电池电动势和内阻的测量
- **学习目标**：学习使用伏安法和图像法测量电池的电动势和内阻，掌握电路连接和数据处理方法
- **视频链接**：B站 BV1Hi1NBAEWm
- **AI问答页面**：`ai-question04-answer.html`

## 使用方法

1. **访问主页**
   - 打开 `index.html` 文件
   - 浏览四个实验卡片

2. **观看完整视频**
   - 点击"观看完整视频"按钮
   - 跳转到B站观看完整实验视频

3. **使用AI问答**
   - 点击"进入AI问答"按钮
   - 在搜索框输入问题（如"如何搭建实验装置？"）
   - 点击"AI搜索视频片段"按钮
   - 查看匹配的视频片段，点击"播放片段"观看

4. **常见问题示例**
   - 每个AI问答页面提供常见问题示例
   - 点击示例问题快速填充搜索框

## 技术特点

- **纯前端实现**：无需后端服务器，可直接在浏览器中运行
- **语义匹配算法**：
  - 关键词匹配
  - 步骤匹配（step字段）
  - 关键点匹配（keyPoints字段）
  - 连续文字匹配
  - 短语模式匹配
- **视频集成**：支持B站视频嵌入播放
- **现代化UI**：使用渐变色彩和卡片式设计
- **Font Awesome图标**：丰富的图标支持

## 注意事项

- 建议登录B站账号以获得最佳观看体验
- 电脑端视频默认为静音，请点击视频右下角音量按钮解除静音
- 建议先完整观看视频，再使用AI问答功能查找具体问题
